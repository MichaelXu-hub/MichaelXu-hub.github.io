CorelDraw 2019

CorelDRAW 2019 是 Corel 公司在 2018 年发布的一个重要版本（通常简称 CorelDRAW 2019），它是这款历史悠久、功能强大的矢量图形设计软件的一个里程碑式更新。它不仅是 CorelDRAW 诞生 30 周年的献礼版本，也引入了多项旨在提升工作效率和创造力的新功能。

软件功能

1、性能和兼容性提升
原生支持 macOS： 这是最大的亮点之一。CorelDRAW 2019 提供了专为 Mac 系统优化的原生版本，而不再是通过虚拟机运行，这使得 Mac 用户可以获得更流畅、更稳定的体验。

大幅提升的启动速度和文件打开速度： 软件的整体响应速度得到了优化。

支持 Microsoft Windows 10： 完全兼容最新的 Windows 系统。

2、创新设计工具
对称绘图模式： 允许用户在绘制时实时创建对称的图案和形状。可以设置多种对称模式（如水平、垂直、双轴对称、径向对称），非常适合创作曼陀罗、花纹、角色设计等。

点阵图效果遮罩： 提供了一个非破坏性的方式，将点阵图（位图）效果作为遮罩应用于矢量对象或群组上，可以轻松创建复杂的纹理和效果，并且随时可编辑。

图块阴影工具： 可以快速为对象或文本添加逼真的阴影效果，操作简单直观，比传统的阴影工具更灵活。

增强的矢量预览功能： 在编辑复杂图形时，可以隐藏对象的填充，只显示轮廓，大大提高了编辑的流畅度。

3、协作与输出功能
支持 Microsoft Surface Dial： 在支持 Surface Dial 的设备上，用户可以直接通过旋转 Dial 来调整笔触大小、颜色、缩放等参数，提升创作效率。

发布至 WordPress： 可以直接将设计好的图片导出并上传到 WordPress 网站，方便博客作者和网站管理员。

改进的 PDF 支持： 提供更强大、更可靠的 PDF/X 支持，确保打印输出的准确性。

4、学习资源和用户体验
内置学习资源： 软件内提供了大量的提示、教程和视频内容，帮助新用户快速上手。

新的欢迎屏幕： 界面更加现代化，可以快速访问最近的文件、模板和学习资源。
