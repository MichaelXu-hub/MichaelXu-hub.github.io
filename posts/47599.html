SQL Server 2012

SQL Server 是一个功能全面、安全可靠、性能卓越的企业级数据平台。它既适合运行传统的业务应用（OLTP），也擅长进行复杂的数据分析和商业智能（OLAP），并且通过与微软 Azure 云的深度集成，为现代混合云架构提供了强大的支持。提供了从数据存储、管理到高级数据分析的一整套解决方案。

软件功能

1、数据存储与管理：

表 (Table)： 以行和列的结构化方式存储数据。

视图 (View)： 虚拟表，简化复杂查询，增强数据安全性。

存储过程 (Stored Procedure)： 预编译的 T-SQL 代码块，提高执行效率，减少网络流量。

函数 (Function)： 可重用的代码逻辑，用于计算并返回值。

触发器 (Trigger)： 自动响应特定数据库事件（如 INSERT, UPDATE）的特殊存储过程。

2、强大的查询语言 - T-SQL：

SQL Server 使用 Transact-SQL (T-SQL)，这是对标准 SQL 的扩展，增加了变量、循环、异常处理等编程功能，使其功能更强大。

3、高可用性 (High Availability)：

Always On 可用性组： 企业级功能，提供数据库镜像和故障转移集群的结合，实现跨服务器的高可用和灾难恢复。

数据库镜像 & 日志传送： 提供数据库级别的冗余和故障转移（较旧的技术，正逐渐被 Always On 取代）。

故障转移集群： 在服务器实例级别提供高可用性（Windows Cluster 功能）。

4、安全性 (Security)：

精细的权限控制： 从服务器登录、数据库用户到架构和对象级别（如表、列）的权限管理。

数据加密： 支持透明数据加密（TDE）、列级加密和备份加密，保护静态数据和传输中数据的安全。

动态数据掩码： 对非授权用户隐藏敏感数据（如只显示身份证号后四位）。

行级安全性： 控制用户只能访问表中特定的行数据。

安装教程

1.选中下载的【SQL Server 2012 64位】压缩包，鼠标右击选择【解压到SQL Server 2012 64位】。

2.双击打开解压后的【SQL Server 2012 64位】文件夹。

3.鼠标双击打开【cn_sql_server_2012_x64.iso】。

4.选中【setup.exe】可执行文件，鼠标右击选择【以管理员身份运行】。

5.点击【安装】，然后点击【全新SQL Server 独立安装或向现有安装添加功能】。

6.等待进度条走完，点击【确定】。

7.选择【输入产品密钥】，输入以下密钥。密钥输入第一个不行就试试第二个。再点击【下一步】。

【FH666-Y346V-7XFQ3-V69JM-RHW28】

【YFC4R-BRRWB-TVP9Y-6WJQ9-MCJQ7】

8.选择【我接受许可条款】和【将功能使用情况……..】。然后点击【下一步】。

9.等待状态出现【已完成】。会自动进入下一个页面。

10.点击【下一步】。

11.选择【SQL Server功能安装】。然后点击【下一步】。

12.点击【全选】，更改软件的安装路径和共享目录。建议将软件安装到除C盘以外的磁盘，我这里安装到D盘，所以把原路径中的C改为D即可，然后点击【下一步】。

13.等待进度条走完。然后点击【下一步】。

14.选择【默认实例】。建议把SQL Server软件实例根目录安装到除C盘以外的磁盘，我这里安装到D盘，所以把原路径中的C改为D即可，然后点击【下一步】。

15.点击【下一步】。

16.点击【下一步】。

17.选择【windows 身份验证模式】，点击【添加当前用户】。然后点击【下一步】。

18.选择【多维和数据挖掘模式】，点击【添加当前用户】。然后点击【下一步】。

19.点击【下一步】。

20.点击【添加当前用户】。然后点击【下一步】。

21.建议把SQL Server软件工作目录和结果目录安装到除C盘以外的磁盘，我这里默认安装到D盘。点击【下一步】。

22.点击【下一步】。

23.点击【下一步】。

24.点击【安装】。

25.软件正在安装中，稍等一会儿。

26.安装成功，点击【关闭】。

27.点击右上角的【×】关闭该页面。

28.点击左下角的【开始菜单】，找到【SQL Server Management Studio】，按住鼠标左键往桌面拖动即可创建桌面快捷方式。

29.在桌面双击打开【SQL Server Management Studio】软件图标运行软件。

30.点击【连接】。

31.软件安装完成。
